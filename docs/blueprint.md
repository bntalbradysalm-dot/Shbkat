# **App Name**: Shabakat Wallet

## Core Features:

- Balance Display: Display the user's current balance in Yemeni Rial.
- Account Funding: Allow users to fund their accounts.
- Wallet System: Integration with Al-Wadi system
- Network Services: Provide access to various network services.
- Favorites: Allow users to save favorite transactions or contacts.
- Transfer to Subscriber: Enable users to transfer funds to other subscribers.
- Customer support: Offer user to use customer support service

## Style Guidelines:

- Primary color: Vibrant teal (#00A693) to represent financial well-being and trustworthiness.
- Background color: Light gray (#F5F5F5), almost white, to provide a clean and modern backdrop.
- Accent color: Soft green (#90EE90), a lighter and desaturated tone from teal, to highlight key actions and elements without overpowering the primary color.
- Font: 'PT Sans' for both body and headlines, a humanist sans-serif to maintain a modern and clean feel while offering readability and approachability.
- Use simple, outlined icons with rounded edges, colored in teal to maintain consistency and a modern aesthetic.
- Employ a clean, card-based layout with ample spacing and consistent padding to improve readability and user experience.
- Subtle animations for transactions and balance updates to provide visual feedback and enhance user engagement.
